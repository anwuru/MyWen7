## E Add Modulo 10

> 02:11 (+2)
>
> 交了两次都是错的，最后就改对了，很可能是 0 5 10 15 的讨论

```cpp
	int a;
    cin >> a;
    int i = a % 20;
    if (i == 5 || i == 15) {
      a += 5;
      i = (i + 5) % 20;
    }
```

就加了这个 `if`。
